# Posix Command Line Interface

- A simple, rewritten version of the CLI for Unix Systems.

- For COMP3370

## Commands:

- exit
    - Args: None
- cd
    - Args: `cd <path>`
- ls
    - Args: `ls`
- ps
    - Args: None
- kill
    - Args: `kill <pid>`
- clear:
    - Args: None
